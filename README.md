# Don't Crash

## Environment
- Python 2.6, 2.7 or 3.3+
- Cocos2d
```
pip install cocos2d
```
- AVBin
> Link: http://avbin.github.com/AVbin/Download.html  
the Mac can't import pyglet.media.avbin, follow this link to resolve it.


## Install
```
git clone https://github.com/FloydaGitHub/DontCrash.git
cd DontCrash
make
```
